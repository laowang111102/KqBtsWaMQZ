# 社团管理系统 java463

## 项目概述

社团管理系统是基于Java技术开发的，适用于各类社团的组织与管理工作。系统采用B/S架构和MVC模式，支持多平台运行，提供完整的前后端代码、详细文档及部署指南。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：IDEA、JDK 1.8、Maven、MySQL
- 技术栈：Java、MySQL、SpringBoot、Mybatis、Vue等

## 功能模块

### 系统角色

- 管理员
- 团长
- 学生

#### 管理员模块

- 首页
- 学生信息管理
- 社团信息管理
- 团长信息管理
- 活动信息管理
- 公告信息管理

#### 团长模块

- 功能模块（具体功能详见文档）

## 运行环境

- 服务器：建议使用Java兼容的服务器，如Tomcat
- 数据库：MySQL
- 客户端：支持现代浏览器

## 部署步骤

1. 环境搭建：配置Java开发环境及MySQL数据库
2. 数据库初始化：执行提供的SQL脚本进行数据库的初始化
3. 后端部署：使用IDEA或Maven构建后端项目，并部署到服务器
4. 前端部署：将前端代码放置到Web服务器目录下

## 目录结构

```
项目根目录/
├── backend                # 后端代码目录
├── frontend               # 前端代码目录
├── document               # 项目文档目录
└── sql                    # 数据库初始化SQL脚本目录
```
